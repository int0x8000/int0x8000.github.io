# about

By work I'm at an MSP leading a Microsoft Intune + Entra ID rollout across multiple client tenants — 1,000+ Windows endpoints, hybrid Azure AD, Autopilot, Win32 app packaging, the works. It's the kind of project most people at my level don't get anywhere near, and I've learned more from the edge cases and gotchas than any cert course.

At home I'm running a Dell PowerEdge R630 out of my home — Proxmox, 34 LXC containers, Graylog collecting logs from all of them, Wazuh watching for threats, Ansible managing the fleet, and a Detection-as-Code pipeline pushing Sigma rules through GitHub Actions CI/CD. Every project is documented like I'm handing it off to someone else, because I think that's what separates engineers from technicians.

I'm looking to make the jump into a dedicated Systems Engineering, IAM, or Security Operations role where I can do this kind of work full time.
